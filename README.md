# L'Atelier des Sens

A luxury perfume landing page featuring immersive WebGL fluid effects, interactive scratch cards, and smooth scroll animations.

## Features

- **WebGL Fluid Background** — Interactive Three.js shader that reacts to mouse movement
- **Scratch Cards** — Interactive canvas elements that reveal hidden text
- **Product Showcase** — 3D tilt effect on hover for perfume bottles
- **Floating Ingredients** — Animated text elements with particle canvas background
- **Custom Cursor** — Gold cursor with trail effect
- **Smooth Scrolling** — Lenis-powered smooth scroll with GSAP animations

## Tech Stack

- React + TypeScript + Vite
- Three.js / React Three Fiber
- GSAP + ScrollTrigger
- Lenis (smooth scroll)
- Tailwind CSS

## Sections

1. **Hero** — Full-screen with animated fluid shader background
2. **Philosophy** — Interactive scratch cards revealing brand values
3. **Collections** — Men's and women's perfume showcase
4. **Atelier** — Ingredient visualizer with particle effects
5. **Contact** — CTA and footer
